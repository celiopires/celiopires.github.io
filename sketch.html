Sketch

Started using Sketch in the year of 2014. Since then this software has got a lot of improvements and I can assure you that designing interfaces has never been so fast and pleasant.

Maintaining a design consistency on all platforms made me explore in deep what this tool has to offer. I find it very useful especially when it comes to implementing new UI elements to different screen sizes. I’ve made my first Design System using out most of Sketch and the results were just stunning. Symbols, plugins and layouts are probably the tools I use the most.
